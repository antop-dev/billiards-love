==== 경기 조회

한 경기^`match`^에 대한 정보를 조회 한다.

현재 로그인한 회원에 따라서 상대방이 결정된다.

예) A 참가자와 B 참가자가 경기하는 경기를 조회할 경우

* A 참가자가 조회 시 상대방은 B가 된다.
* B 참가자가 조회 시 상대방은 A가 된다.

``closed`` 속성이 ``true``이면 알반 참가자는 UI에서 수정이 불가능하다.

|===
|코드|내부 값|설명

|ACCEPTING
|1
|접수중 (시작하지 않음)

|2
|준비중
